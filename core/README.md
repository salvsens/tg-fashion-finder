# Core

Ядро системы: управление базой данных.

### Схема БД (`items`)
Хранит данные о найденных товарах:
* **`id`**: INTEGER PRIMARY KEY AUTOINCREMENT — уникальный идентификатор записи.
* **`source_channel`**: TEXT — название и ссылка на первоисточник.
* **`raw_text`**: TEXT — исходный текст поста (база для поиска).
* **`image_url`**: TEXT — прямая ссылка на изображение товара.
* **`price`**: INTEGER DEFAULT 0 — распознанная цена айтема.
* **`category`**: TEXT DEFAULT 'uncategorized' — категория (обувь, верхняя одежда и т.д.).
* **`is_processed`**: BOOLEAN DEFAULT 0 — флаг обработки (0 — новый, 1 — обработан ML).

### Основные функции:
* **`init_db()`**: Инициализация таблиц при первом запуске.
* **`add_raw_post()`**: Сохранение нового поста из парсера (Crawler).
* **`get_unprocessed_posts()`**: Выборка новых данных для анализа ML-частью.
* **`update_post_data()`**: Обогащение записи ценой и категорией после обработки.
* **`search_items()`**: Поиск по базе с фильтрацией на Python (Case-insensitive).
