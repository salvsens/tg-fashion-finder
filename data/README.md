# Схема данных и миграции

В этой папке хранятся SQL-скрипты, миграции и настройки базы данных.

### Состав:
* `init.sql`: Скрипт инициализации таблиц (Users, Channels, Products).
* `migrations/`: История изменений структуры БД.
* `ER-diagram.png`: Визуальная схема связей данных.

### Стек:
* `PostgreSQL`
* `Redis` (для кэширования состояний бота)
