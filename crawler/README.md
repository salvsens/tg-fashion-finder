# Модуль парсинга (Crawler)

Этот модуль отвечает за сбор данных и поиск новых источников через Telegram MTProto API.

### Функционал:
* **Discovery:** Использование метода `getChannelRecommendations` для автоматического поиска каналов по продаже одежды.
* **Scraping:** Асинхронный сбор постов из целевых каналов с помощью библиотеки `Telethon`.
* **Data Cleaning:** Первичная фильтрация мусорных постов и дублей.

### Технологии:
* `Telethon` (MTProto API)
* `Asyncio`

**Разработчик:** Бутенко Владислав
