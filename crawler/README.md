# Модуль парсинга (Crawler)

Автоматический сбор контента из Telegram-каналов через Bot API.

### Функционал:
* **Фильтрация:** Проверка постов на соответствие «белому списку» (Whitelist) по ID.
* **Парсинг:** Сбор текста, подписей и file_id изображений.
* **Логирование:** Формирование читаемых имен каналов и запись в БД.

### Технологии:
* `aiogram 3.x` (Telegram Bot API)
* `Python / Asyncio`
* `PostgreSQL` (через модуль database)
