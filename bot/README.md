# Интерфейс пользователя (Telegram Bot)

Фронтенд-часть проекта, реализованная в виде Telegram-бота для взаимодействия с конечным пользователем.

### Функционал:
* **Search Interface:** Удобный поиск товаров по базе данных с фильтрами.
* **User Onboarding:** Регистрация предпочтений пользователя (размеры, стили).
* **Delivery:** Отображение карточек товаров с прямыми ссылками на посты-источники.

### Технологии:
* `Aiogram 3.x` (Bot API)
* `Pydantic` (валидация данных)
