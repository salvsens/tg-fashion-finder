# Модуль Telegram-бота (`/bot`)

Данная директория содержит программный код интерфейса взаимодействия с пользователем.

### Структура файлов:
* **`main.py`**: Точка входа в приложение и основная логика обработки команд.
* **`__init__.py`**: Инициализация модуля.

### Ключевые функции:
* **`cmd_start`**: Обработка стартовой команды, инициализация сессии пользователя.
* **`cmd_search`**: Интерфейс поиска товаров по базе данных с выводом карточек.
* **`get_like_keyboard`**: Генерация динамических `Inline`-клавиатур для системы лайков.
* **`process_like`**: Callback-хендлер для обработки нажатий на кнопки «В избранное».
* **`main`**: Асинхронная функция запуска (polling) и конфигурация диспетчера.

### Константы и конфигурация:
* **`BOT_TOKEN`**: Загрузка токена из переменных окружения.
* **`bot`**: Объект клиента Telegram Bot API.
* **`dp`**: Диспетчер для регистрации роутеров и хендлеров.
