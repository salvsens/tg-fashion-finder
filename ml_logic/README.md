# Модуль интеллектуальной обработки (ML Logic)

Модуль для анализа текстового и медиа-контента, собранного парсером.

### Функционал:
* **Extraction:** Извлечение цены, бренда и категории товара из описания.
* **Semantic Search:** Реализация поиска с учетом синонимов (например, поиск "куртки" по запросу "пуховик").
* **Classification:** Сортировка товаров по категориям (обувь, аксессуары, верхняя одежда).

### Технологии:
* `Sentence-Transformers` / `NLTK`
* `PostgreSQL Full-Text Search`
