# Модуль обработки данных (ML Logic)

Модуль для автоматического анализа и структурирования контента, собранного парсером. Отвечает за конвертацию неструктурированного текста в атрибуты товара.

### Ключевой функционал:
* Price Extraction — Извлечение цены из текста с использованием регулярных выражений (RegEx), поддержка различных форматов валют.
* Category Detection — Классификация товаров по ключевым словам на основе весового сопоставления.
* Batch Processing — Асинхронная обработка очереди записей из базы данных.

### Технологии и методы:
* Python 3.11+
* RegEx (библиотека re) — Паттерны для поиска числовых данных.
* Asyncio — Интеграция с асинхронным ядром базы данных.
* Keywords Mapping — Алгоритм сопоставления категорий на основе лингвистических маркеров.

### Основные функции:
* extract_price(text) — Парсинг цены с учетом контекста и фильтрацией аномалий.
* detect_category(text) — Определение категории одежды по словарям соответствия.
* process_posts() — Главный цикл-обработчик для взаимодействия с core/database.py.
